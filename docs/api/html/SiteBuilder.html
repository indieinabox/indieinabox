SiteBuilder

Namespace: Indieinabox

Class SiteBuilder

Orchestrates the static site generation process. It scans the content directory, virtualizes missing translations, processes markdown into HTML/Gemtext/Gophermap, and compiles feeds and assets into the output directory.
